# Gemography Frontend

## Features Implemented
```
-Pagination
-Infinite Loading
-Fetching API Endpoints
-Results in the last 30 days (relative to 2017-11-22)
```

### Technologies used
```
-CSS
-TailwindCSS
-Vue.JS (Vue2)
-Axios
```

### Packages used
```
-Axios for vue2
-Vue Infinite Loading

```

#### To Run this project locally use ``npm run serve``
